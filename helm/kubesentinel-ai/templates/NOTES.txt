  3. Prometheus / Loki:        {{ .Values.collector.prometheusUrl | default "<미설정>" }} / {{ .Values.collector.lokiUrl | default "<미설정>" }}
  4. Notifier:                 type={{ .Values.notifier.type }} {{ if .Values.secret.notifierWebhook }}(webhook 설정됨){{ else }}(webhook 미설정 → 알림 비활성/noop){{ end }}
